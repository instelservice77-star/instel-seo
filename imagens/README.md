# Imagens do Pipeline SEO — Instel Service

Adicione aqui as fotos dos seus serviços em formato `.webp` com os nomes exatos abaixo.
Na primeira execução do workflow, elas serão enviadas automaticamente ao WordPress.

## Arquivos esperados

| Arquivo | Sugestão de foto |
|---------|-----------------|
| eletrica-01.webp | Engenheiro inspecionando painel elétrico |
| eletrica-02.webp | Quadro de distribuição com disjuntores |
| eletrica-03.webp | Medição com alicate amperímetro |
| eletrica-04.webp | Fiação com inconformidade marcada |
| eletrica-05.webp | Medição de aterramento elétrico |
| eletrica-06.webp | Instalação em área industrial/classificada |
| eletrica-07.webp | Laudo ou relatório técnico impresso |
| eletrica-08.webp | Tomada ou ponto elétrico em vistoria |
| eletrica-09.webp | SPDA / para-raios em edificação |
| eletrica-10.webp | Medidor de tensão em instalação |

## Como converter para .webp

No Mac, use o Preview: Arquivo → Exportar → formato WebP.  
Ou use o site gratuito: https://squoosh.app

## Dimensões recomendadas

800×533 px (proporção 3:2), tamanho máximo 200 KB por arquivo.
